# Study-Python-Cpp-Repo
Учебный репозиторий для совмещения кода на C++ и Python
## Идея
Идея проекта заключается в создании программы на Python, который в процессе работы будет использовать скрипты на C++, что сильно укоротит время работы программы. Для примера была взята программа, которая должна посчитать количество каждого слова в текстовом файле. Не трудно догататься, что скрипт на Python будет считать это в разы медленнее, чем C++.
## Реализация проекта
Главный файл main.py запускается из консоли и принимает аргументы: название исходного файла (по умолчанию - txt/Master_and_Margarita.txt), название файла для результатов (по умолчанию - result.json), также аргумент --total_count добавляет в результат общее число слов. Результат записывается в виде json, отсортированный в лексикографическом порядке. В процессе работы, main.py вызывает скомпилированные скрипты на C++ с помощью команды:
``` os.system("./exe/$СКРИПТ$") ```
## Структура проекта
* main.py - главный файл, запускающий работу программы;
* src/ - папка с кодом на C++;
* exe/ - папка с собранными исполняемыми файлами;
* txt/ - папка с текстовыми файлами для проверки работы программы.
## Требования при установке
* На устройстве должен быть установлен компилятор clang++ любой версии
