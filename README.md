# Study-Python-C-Repo
Учебный репозиторий для совмещения кода на C++ и Python
## Идея
Идея проекта заключается в создании программы на Python, который в процессе работы будет использовать скрипты на C++, что сильно укоротит время работы программы. Для примера была взята программа, которая должна посчитать количество каждого слова в текстовом файле. Не трудно догататься, что скрипт на Python будет считать это в разы медленнее, чем C++.
## Реализация проекта
Главный файл main.py запускается из консоли и принимает аргументы: название исходного файла (по умолчанию - txt/Master_and_Margarita.txt), название файла для результатов (по умолчанию - result.json), также аргумент --total_count добавляет в результат общее число слов. Результат записывается в виде json, отсортированный в лексикографическом порядке.
